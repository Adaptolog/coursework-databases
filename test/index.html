# Тестування працездатності системи

Для тестування системи використовувався Swagger, який дозволяє зручно тестувати всі ендпоінти API через браузер. Swagger інтегровано в проект за допомогою бібліотеки Swashbuckle.AspNetCore, що автоматично генерує документацію та інтерфейс для тестування API.
